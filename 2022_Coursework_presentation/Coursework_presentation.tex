	\section{Висновки}
	\begin{frame}
	\begin{itemize}
		\item Стеганографія в поєднанні з криптографією, є потужним захистом інформації.
		\item Метою стеганографічних алгоритмів є забезпечити  приховування факту наявності даних, що потребують захисту.
		\item Надійними алгоритмами захисту даних в зображеннях є алгоритми маскування та перетворення.
		\item Найбільш надійним для захисту даних в текстових форматах є метод знаків одного накреслення.
		\item Для аудіо-форматів найефективнішим вважають метод заміни найменш значущого біта. 
		\item Метод ЗНЗБ для приховування даних в BMP форматі є простим та може приховувати великий об'єм даних.
		\item Якщо збільшити розмір вбудованих даних, то погіршиться якість зображення.
	\end{itemize}
	\end{frame}
