	\subsection{Приховування інформації в аудіосигналах}
	\begin{frame}
	Вимоги до стегоситеми:
	\begin{itemize}
		\item стійкість даних до фільтрації, різних шумів, втрат зі стистом, аналогових та цифрових модифікацій;
		\item у випадку ЦВДЗ бажання позбутись даних може пошкодити контейнер;
		\item дані не мають докладати в сигнал спотворення помітні людині;
		\item дані  не мають значно впливати на сатистику контейнера.
	\end{itemize}
	Методів приховування інформації в аудіосигналах:
	\begin{itemize}
	\item заміни найменш значущого біта;
	\item фазового кодування;
	\item розширення спектру;
	\item приховування інформації за допомогою вставки тонів;
	\item приховування інформації використовуючи ехо-сигнал.
	\end{itemize}
	\end{frame}

	\subsection{Класифікація атак на стегоситеми}
	\begin{frame}
	Види атак на стегосистему, грунтуються на:
	\begin{itemize}
		\item знанні заповненого контейнера;
		\item відомому повідомленні;
		\item прихованих даних чи заповнених контейнерів, що обирались зловмисником;
		\item відомому пустому контейнері;
		\item  виборі порожнього контейнера зловмисником;
		\item на знанні математичної моделі контейнера.
	\end{itemize}
	Етапи зламування стегосистеми:
	\begin{itemize}
		\item розкриття факту існування прихованої інформації;
		\item добування прихованих даних;
		\item перетворення секретних даних;
		\item заборона поширення секретного повідомлення.
	\end{itemize}
	\end{frame}

	\item використовуючи ключ, вибрати найменш значущі біти кольорів пікселів та згенерувати з них секретні дані.

	 \subsection{Метод \mbox{Коха–Жао} в JPEG форматі}
	\begin{frame}
	 Алгоритм Коха-Жао для побудови інформації, що використовує частотну область контейнера, полягає у відповідному зміні коефіцієнта дискретного косинусного перетворення (ДКП). Зображення розбивається на блокі розміром 8$\times$8 пікселів і до кожного блоку застосовується ДКП. Кожен блок може вміщувати для запису один біт інформації.\\[3mm]
	 
	  Внесення змін в коефіцієнти спотворює початкове зображення за умови, що відносна величина не відповідає засекреченому біту. Зі збільшенням цього значення стегосистема стає стійкішою, проте меншою стає якість зображення. \\[3mm]
	 
	  Для атак на даний алгоритм потрібно визначити пари коефіцієнтів дискретно косинусного перетворення та порогові значення.
	\end{frame}

	\section{Висновки}
	\begin{frame}
	\begin{itemize}
		\item Стеганографія в поєднанні з криптографією, є потужним захистом інформації.
		\item Метою стеганографічних алгоритмів є забезпечити  приховування факту наявності даних, що потребують захисту.
		\item Надійним алгоритмом захисту даних в зображеннях є алгорит маскування.
		\item Найбільш надійним для захисту даних в текстових форматах є метод знаків одного накреслення.
		\item Для аудіо-форматів найефективнішим вважають метод заміни найменш значущого біта. 
		\item Метод ЗНЗБ для приховування даних в BMP форматі є простим та може приховувати великий об'єм даних.
		\item Збільшуючи розмір вбудованих даних зменшується якість зображення.
		\item Метод Коха-Жао є стійким до більшості відомих стегоатак, але, має низьку пропускну здатність
	\end{itemize}
	\end{frame}
